# 🎓 Udemy Courses Data Analysis

## 📌 Project Overview
This project focuses on analyzing Udemy courses dataset to extract meaningful insights using Python.  
The goal is to perform Exploratory Data Analysis (EDA), clean the data, and generate business insights that can help understand course performance.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Handle missing values
- Conduct Exploratory Data Analysis (EDA)
- Analyze course prices, ratings, subscribers, and categories
- Extract actionable business insights
- Create visualizations for better understanding

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Key Analysis Performed
- Distribution of course prices
- Most popular course categories
- Top rated courses
- Relationship between price and number of subscribers
- Free vs Paid course comparison
- Instructor performance analysis
